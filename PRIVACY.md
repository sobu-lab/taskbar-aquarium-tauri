# Privacy Policy / プライバシーポリシー

Last updated / 最終更新: 2026-06-07

## English

**Taskbar Aquarium** does not collect, store, or transmit any personal information.

### Data collection
This app does not collect any user data, including but not limited to:
- Personally identifiable information (name, email, address, etc.)
- Usage analytics or telemetry
- Crash reports
- Location data
- Device identifiers

### Local data storage
The app stores only the following settings locally on your device in `%APPDATA%\com.sobu-lab.taskbar-aquarium-tauri\config.json`:
- Window position and width
- Number of fish
- Pixel size
- Background transparency setting

This data is never transmitted off your device.

### Network communication
The app does not make any network requests. It works completely offline.

### Third-party components
The app uses Microsoft WebView2 to render its UI. WebView2 is part of Microsoft Edge and may have its own telemetry governed by Microsoft's privacy policy: https://privacy.microsoft.com/

### Contact
For questions about this policy, please open an issue at:
https://github.com/sobu-lab/taskbar-aquarium-tauri/issues

## 日本語

**Taskbar Aquarium** は、いかなる個人情報も収集・保存・送信しません。

### データ収集について
本アプリは以下を含む、ユーザーデータを一切収集しません：
- 個人を特定できる情報（氏名、メールアドレス、住所など）
- 利用状況の分析データやテレメトリ
- クラッシュレポート
- 位置情報
- デバイス識別子

### ローカル保存データ
本アプリは、お使いの端末の `%APPDATA%\com.sobu-lab.taskbar-aquarium-tauri\config.json` に以下の設定のみを保存します：
- ウィンドウの位置と幅
- 魚の数
- ピクセルサイズ
- 背景透過の設定

これらのデータは端末外に送信されることはありません。

### ネットワーク通信
本アプリはネットワーク通信を一切行いません。完全にオフラインで動作します。

### サードパーティ コンポーネント
本アプリは UI 描画に Microsoft WebView2 を使用しています。WebView2 は Microsoft Edge の一部であり、独自のテレメトリを持つ場合があります。詳細は Microsoft のプライバシー ポリシーをご参照ください：https://privacy.microsoft.com/

### お問い合わせ
本ポリシーに関するご質問は、GitHub の Issue にてお願いいたします：
https://github.com/sobu-lab/taskbar-aquarium-tauri/issues
